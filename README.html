避免生产环境暴露如下 collections:

  • resumes
  • diary

jekyll 的生产环境判断受控于环境变量，确保 npm run build 有指定这个环境变量（见 package.json 和 deploy-workflow.yml）：

JEKYLL_ENV=production

具体控制的方法是通过 _config_dev.yml 覆盖 _config.yml, 确保 npm start 执行如下方法启动 jekyll:

bundle exec jekyll build --watch --config _config.yml,_config_dev.yml
