使用 react-devtools

网络

Network layer in React Native – DailyJS – Medium

RN 提供三个网络 APIs

  1. fetch
  2. XMLHttpRequests
  3. WebSocket

fetch XMLHttpRequest，后端都是基于 Networking，Android 由 OkHttp 实现，iOS 基于 URLSession

WebSockets Android 同样使用 OkHttp，iOS 基于 NSStream

在 RN 上使用 axios，会使用 XHR，见 axios/defaults.js at master · axios/axios

Components

props

props 用于标签属性

class Greeting extends Component {
  render() {
    return (
      <Text>Hello {this.props.name}!</Text>
    );
  }
}

使用 ：<Greeting name='Rexxar' />

  • state

用于数据绑定的量，state 变化会触发 UI 更新，一般在 constructor 初始化。

constructor(props) {
    super(props);
    this.state = {isShowingText: true};
}

访问 this.state.isShowingText
修改 this.setState(preState => { return {isShowingText: !preState.isShowingText}})
