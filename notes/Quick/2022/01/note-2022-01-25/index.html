YUV

类似于 RGB 的另外一种表述颜色的方法

  • 中 Y 表示明亮度（Luminance、Luma），也就是灰阶值。
  • U、V 表示色度（Chrominance 或 Chroma），描述的是色调和饱和度。

YUV 采样

  • 可以理解为将多个 YUV 像素表示图像，
  • YUV 所表示的图像，Y 和 UV 分量是分离，解决与和黑白电视机的兼容问题
  • 人眼对色度的敏感程度低于对亮度的敏感程度，所以可以通过几个 Y 分量共用 UV 分量来节约空间

采样格式：

  • YUV444，每个像素都用 Y、U、V 三分量表示，单个像素占用 3 个字节：Y0U0V0
  • YUV422，UV 分量的采样率是 Y 分量的一半（间隔采样），两个像素占用 4 个字节：Y0U0Y1V0
  • YUV420，一行(固定步进)只间隔采样 U 分量，另一行只间隔采样 V 分量，8 个像素 12 字节：Y0U0Y1Y2U2Y3 Y4V4Y5Y6V6Y7

存储方式

  • 平面（planar format），各个分量独立连续存储
  • 打包（packed format），按像素点连续存储，具体在内存中就是 YUV 交错存储

不同的采样格式和存储方式可以产生出多种存储格式：

  • YUYV， YUV422 采样，打包存储。Y 分量，然后同时间隔采样UV，Y0U0Y1V0 Y2U2Y3V2
  • UYVY，同 YUYV ，顺序不同。U0Y0V0Y1 U2Y2V2Y3
  • YUV422P, YUV422 平面存储，
  • YUV420P, YUV420 平面存储
    • YU12, YUV 顺序存储平面
    • YU21, YVU 顺序存储平面
  • YUV420SP, YUV420 平面存储，Y 单独一个平面，UV 打包存储
    • NV12, Y 平面 UV 交替存储。iOS 格式
    • NV21, Y 平面 VU 交替存储。Android 格式 android.graphics.YuvImage 只支持这个格式和 YUY2(YUYV)

Android Image

android.media.Image MediaCodec 和 CameraDevice 用的图像缓冲

Image.Plane 平面模式存储中的平面对象。除了存储像素数据外，还封装了像素步进（PixelStride）和行步进（RowStride）。 以 YUV420_888 格式为例，Y 平面像素步进为 1 ，表示连续存储。UV 平面像素步进为2，表示间隔存储。行步进就是图像的宽度。

另外 Android 表示格式的有 android.graphics.ImageFormat 和 android.graphics.PixelFormat

对于 YUV 转 RGB android 的支持非常有限，自己实现的话 kotlin 位运算真是让人不适。

fun main() {
    val a:Int = 0xaa
    val i:Int = -16777216.or(a)
    val j:Int = -16777216 or a
    println(Integer.toHexString(i))  // feffff56
    println(Integer.toHexString(j))  // ff0000aa
}

  1. JavaAPI

    YUV420P -> NV21 -> RGB

    YUV420 是 CameraX 获取到的帧格式， NV21 是 YuvImage 对应的 YUV420 采样的格式。 YuvImage 通过 compressToJpeg 将数据转换为 jpeg，再通过 BitmapFactory 将 jpeg 读到 Bitmap

    而且会有厌烦的 onFlyCompress logcat 输出 [core/jni/android/graphics/YuvToJpegEncoder.cpp - platform/frameworks/base.git - Git at Google](https://android.googlesource.com/platform/frameworks/base.git/+/android-4.3_r2/core/jni/android/graphics/YuvToJpegEncoder.cpp#73)

  2. RenderScript

    内置有 android.renderscript.ScriptIntrinsicYuvToRGB 支持。但 RenderScript 在 Android 12 已经弃用。 https://github.com/android/camera-samples/blob/master/CameraUtils/lib/src/main/java/com/example/android/camera/utils/YuvToRgbConverter.kt
